# 🍕 Pizza Sales Dashboard

An interactive sales analytics dashboard built to analyze pizza sales data for the year **January 2015 – December 2015**, providing actionable insights on revenue, order trends, and product performance.

---

## 📊 Dashboard Overview

The dashboard is split into two main pages:

### 🏠 Home – Sales Summary
Provides a high-level snapshot of business performance with key KPIs and trend analysis.

### ⭐ Best/Worst Sellers – Product Performance
Breaks down individual pizza performance by revenue, quantity, and total orders.

## 📈 Insights & Findings

### 🗓️ Busiest Days & Times
- Orders are **highest on weekends**, especially **Friday and Saturday evenings**
- Peak months are **July** and **January**

### 🍕 Sales by Category
- **Classic category** contributes the most to total sales and orders
- **Large size** pizzas drive maximum revenue

### 📅 Monthly Trend
- Orders peak in **July (1935)** and dip in **September–October**
- The monthly trend shows a mid-year surge followed by a year-end recovery

### 📆 Daily Trend
- **Friday** is the busiest day with **3.5K orders**
- **Sunday** has the lowest order volume at **2.6K orders**

## 🚀 How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/hariprasanth02/Pizza_Sales_Analysis.git
   ```
2. Open `Pizza_Sales_Report.pbix` in **Power BI Desktop**
3. Load the dataset from the `data/` folder if prompted
4. Use the **Pizza Category** filter and **date slider** to explore different segments
